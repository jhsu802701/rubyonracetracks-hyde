Ruby on Rails

Changing the Time Zone

Go to the config/environments/*.rb files and set config.time_zone to the appropriate value listed at http://api.rubyonrails.org/classes/ActiveSupport/TimeZone.html

Updating Ruby/pg/nokogiri

NOTE: The all.sh script must be added to legacy projects.
    1. Update the Ruby version, pg gem version, and nokogiri gem version in the Gemfile.
    2. Update the Ruby version in the .travis.yml file.
    3. bundle update; sh all.sh

Updating gems

    1. Update the versions in the Gemfile.
    2. bundle update; sh all.sh

Specific Commands for Creating Objects

  • Adding a model:
    • rails generate model Object field1:type1 field2:type2 . . . user:references (if connected to user)
    rails generate model Object field1:type1 field2:type2 . . . (if not connected to user)
  • Adding a controller:
    • rails generate controller Objects new
  • Adding an integration test:
    • rails generate integration_test (name of test)

Examples of Object Methods Available from the Rails Tutorial Sample App

  • Index/show: users
  • New/create: users, microposts, relationships, sessions, password resets
  • Edit/update: users, password resets
  • Destroy/delete: users, microposts, relationships, sessions

General Procedures for Creating Objects

For each of these 7 steps, create a Git branch. Run the all.sh script when you think you're finished. Once you really are finished, merge the Git branch into the master branch.
    1. Model
      • Add model
      • Unit tests
    2. db/seeds.rb
    3. Index
      • Add controller (required even if skipping index)
      • Functional and integration tests
      • config/routes.rb: enable index
      • Controller: def index
      • index.html.erb and link to this page
    4. Show
      • Functional and integration tests
      • config/routes.rb: enable show
      • Controller: def show
      • show.html.erb and link to this page
    5. New/create
      • Functional and integration tests
      • config/routes.rb: enable new and create
      • Controller: def new, def create
      • new.html.erb and link to this page
    6. Edit/update
      • Functional and integration tests
      • config/routes.rb: enable edit and update
      • Controller: def edit, def update (may require @object=statement for both)
      • edit.html.erb and link to this page
    7. Destroy
      • Functional and integration tests
      • config/routes.rb: enable destroy
      • Controller: def destroy
      • Link to this action
